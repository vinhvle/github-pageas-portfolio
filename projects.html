PacMen Exercise

Eye Exercise

https://github.com/vinhvle/EyeExercise.git
The mouse movement event is used to update the position of two "eye" elements displayed in a window that follow the mouse's cursor position on the screen using the mouse events.

Real Time Bus Tracker

https://github.com/vinhvle/RealTimeBusTracker.git
Uses the MBTA bus data to determine stops between Harvard and MIT. An animated marker is used on the map to highlight the bus routes.
